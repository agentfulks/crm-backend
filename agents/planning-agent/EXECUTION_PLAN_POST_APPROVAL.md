### Approval Efficiency Tactics

2. **Template Review Mode** - Focus on personalization quality, not full re-read

- [ ] No obvious errors (typos, wrong name, outdated info)

## Daily Operations Rhythm

### Morning Routine (30 min)
1. Check overnight replies (BDR + VC)
2. Update tracking sheet with new responses
3. Queue follow-ups due today
4. Review today's send list

### Midday Execution (60-90 min)
1. Send scheduled BDR messages (20-25)
2. Send scheduled VC messages (8-12)
3. Quick quality check before hitting send

### End-of-Day (20 min)
1. Log all replies in tracking system
2. Schedule follow-ups for tomorrow
3. Update Trello cards (move to appropriate list)
4. Flag any urgent responses for Lucas
