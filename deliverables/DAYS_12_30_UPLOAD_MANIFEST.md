## Upload Instructions

### Option 1: CSV Import (Recommended)

Each batch folder contains `trello_import.csv` ready for Trello import.

1. Open Trello board: https://trello.com/b/tPkRdYjg/vc-outreach-engine
2. Go to "Daily Queue" list
3. Use Trello's CSV import feature
4. Upload each day's CSV file
5. Cards will be created with proper formatting

### Option 2: Manual Upload Script

```bash
# Run from workspace root
for day in {12..30}; do
  echo "Uploading Day $day..."
  # CSV content available at:
  cat deliverables/day${day}_vc_batch/trello_import.csv
done
```
