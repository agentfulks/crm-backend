## TRELLO API CREDENTIALS — SETUP INSTRUCTIONS

To automate Trello card movements, you need API credentials. Follow these steps:

### Step 1: Get Your API Key

1. Go to: https://trello.com/app-key
2. Log in to your Trello account if not already logged in
3. Copy the "Key" value displayed on the page
4. **Save this somewhere secure** — this is your `TRELLO_API_KEY`

### Step 2: Generate a Token

1. On the same page (https://trello.com/app-key), click the "Token" link
2. Click "Allow" to authorize access
3. Copy the long token string that appears
4. **Save this somewhere secure** — this is your `TRELLO_TOKEN`

### Step 3: Find Your Board ID

1. Open your Trello board in a browser
2. Look at the URL: `https://trello.com/b/XXXXXXX/board-name`
3. Copy the `XXXXXXX` part (7-character code)
4. This is your `TRELLO_BOARD_ID`

### Step 4: Store Credentials Securely

Create a `.env` file in your workspace root:

```bash
# /data/workspace/.env
TRELLO_API_KEY=your_api_key_here
TRELLO_TOKEN=your_token_here
TRELLO_BOARD_ID=your_board_id_here
```

Or export them in your shell:

```bash
export TRELLO_API_KEY="your_api_key_here"
export TRELLO_TOKEN="your_token_here"
export TRELLO_BOARD_ID="your_board_id_here"
```

### Step 5: Verify Setup

```bash
# Test the credentials
curl "https://api.trello.com/1/members/me?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN"
```

If you see your Trello profile data, the credentials are working.

### Security Notes
- Never commit `.env` files to git
- Tokens grant access to your boards — keep them secure
- If a token is compromised, revoke it at: https://trello.com/your/account
