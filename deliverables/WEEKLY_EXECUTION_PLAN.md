## 1. PLAN OVERVIEW

### Philosophy

This plan maximizes meeting probability while respecting time constraints. Built for manual execution without API automation.

**Core Principles:**
1. **Tier 1 First:** Send to best-fit funds immediately
2. **Batch by Day:** Execute in original batch groupings for efficiency
3. **Friday Follow-ups:** Consolidate all follow-up activity one day/week
4. **Buffer Time:** 20% slack built in for replies and research

### Weekly Rhythm

| Day | Focus | Time Budget |
|-----|-------|-------------|
| **Monday** | Review + Tier 1 Sends | 60 min |
| **Tuesday** | Continue Sends | 45 min |
| **Wednesday** | Continue Sends | 45 min |
| **Thursday** | Final Sends + Prep | 45 min |
| **Friday** | Follow-ups + Review | 60 min |
| Weekend | — | — |

## 7. DECISION CHECKPOINTS

### Week 3 Checkpoint (March 20)

**Review:**
- [ ] Tier 1 sends complete? (Target: 20)
- [ ] Tier 2 sends on track? (Target: 30)
- [ ] Reply rate >5%?
- [ ] Any meetings booked?

**Decisions:**
- If reply rate <3%: Review email copy, not targeting
- If 1+ meetings: Continue current pace
- If 0 meetings: Consider pausing Tier 3, focus on Tier 1+2 quality

### Week 6 Checkpoint (April 10)

**Review:**
- [ ] 60+ total sends?
- [ ] 5+ replies?
- [ ] 1+ meetings?
- [ ] Time budget sustainable?

**Decisions:**
- If <3 meetings: Reduce Tier 3, double down on warm intros
- If time budget exceeded: Reduce weekly sends to 8
- If burning out: Take 1-week break, maintain follow-ups only

### Week 9 Checkpoint (May 1)

**Review:**
- [ ] 100+ sends?
- [ ] 10+ replies?
- [ ] 3+ meetings?
- [ ] Process improvements identified?

**Decisions:**
- If 3+ meetings: Continue to Week 12
- If <2 meetings: Consider alternative channels (events, warm intros)
- If deal momentum: Prioritize Tier 3A funds only

### Week 12 Checkpoint (May 24)

**Final Review:**
- [ ] All 150 sends complete?
- [ ] 15+ replies?
- [ ] 3-5 meetings?
- [ ] Investment in progress?

**Next Steps:**
- Continue follow-ups for active conversations
- Archive non-responders after 3 follow-ups
- Plan next campaign or focus on closing
