## Problem Identified

The Maton Trello gateway (`api.maton.ai`) requires **AWS Signature Version 4 authentication**, which is not available in the current environment.

### What I Tried:

1. **Bearer Token Auth** - Failed with "Invalid key=value pair"
2. **Query Parameter Auth** (key, token, api_key, apikey, access_token) - All failed with "Missing Authentication Token"
3. **Header Auth** (x-api-key, api_token, Authorization) - All failed
4. **Basic Auth** - Failed with AWS SigV4 requirements message
5. **Internal Gateway** - No Trello API routes available

### Root Cause:

The Maton gateway is an AWS API Gateway endpoint that requires:
- AWS Access Key ID
- AWS Secret Access Key
- AWS Signature Version 4 signed requests

The `MATON_API_KEY` environment variable appears to be an **API Gateway API key** (for rate limiting/plan tracking), NOT AWS credentials for authentication.

### Gateway Response Headers:
```
x-amzn-errortype: MissingAuthenticationTokenException
access-control-allow-headers: Content-Type,X-Amz-Date,Authorization,X-Api-Key,X-Amz-Security-Token
```
